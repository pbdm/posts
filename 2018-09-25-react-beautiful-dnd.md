# react-beautiful-dnd

Beautiful and accessible drag and drop for lists with React

## 基础概念

### [DragDropContext](https://github.com/atlassian/react-beautiful-dnd#dragdropcontext)

整个可拖拽的页面应该包含在这个组件内

### Droppable

children 必须是一个函数

包含一个 Draggable 的数组

### Draggable

children 必须是一个函数

最基础的拖拽组件, 可在 droppable 内和之间拖拽

# JAVA

## 基本数据类型

| 名称              | 字节    | 位     |其他
|-----------------|-------|-------|--|
| byte\(字节型\)     | 1Byte | 8bit  |
| short\(短整型\)    | 2Byte | 16bit |
| int\(整型\)       | 4Byte | 32bit |
| long\(长整型\)     | 8Byte | 64bit |
| float\(单精度浮点\)  | 4Byte | 32bit |
| double\(双精度浮点\) | 8Byte | 64bit |
| char\(字符\)      | 2Byte | 16bit |0-65535, UTF-16 code unit
| boolean\(布尔\)   | 1Byte | 8bit  |

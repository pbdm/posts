## Puppeteer

* `page.evaluateOnNewDocument` 可用于添加全局函数(载入一个外部 js )
* `page.tap` 用于模拟触摸事件
* [获取 dom 信息](https://github.com/GoogleChrome/puppeteer/issues/438)

模拟设备

```javascript
const devices = require('puppeteer/DeviceDescriptors');
await page.emulate(devices['iPad landscape']);
```

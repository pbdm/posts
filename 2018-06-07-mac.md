## Mac下装gevent(goagent要用的东西...)

`ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"`

`brew install libevent`

`easy_install pip`

`pip install python`

`sudo pip install greenlet`

`sudo pip install gevent`

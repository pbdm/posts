# Database

## Mysql

* `select 1` & `select count(*)`
  * 从性能上来看，`select 1` 要优于 `select count(*)` (`*`当然更耗时间啦)
  * `select 1`返回一个数组，有几个就返回几个1
  * `select count(*)`返回一个数，有几个就返回这个数
  * 所以如果没有值的时候, `select 1` 要返回`null` 而 `select count(*)` 返回 `0`，所以两者不简单的相互替换，优化的时候也要注意

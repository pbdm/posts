# Javascript async

## Promise

### 状态

* Pending (进行中)
* Resolved (已完成,Fulfilled)
* Rejected (已失败)

### 状态改变

* Pending -> Resolved
* Pending -> Rejected

`then(res,rej)`为状态改变时的回调函数, 并返回一个新的Promise实例
`Promise.prototype.catch`方法是`then(null, rejection)`的别名
